# body/cycles/

*Cycles* — here live the bluebooks that *tick at known rates*. Each carries its own cadence ; together they give the body its sense of *what time it is*.

- `heartbeat.bluebook` — the mindstream pulse, fatigue and rhythm.
- `breath.bluebook` — the 0.2Hz inhale-exhale phase flip.
- `circadian.bluebook` — the 24h wall-clock segmenter (dawn → morning → afternoon → dusk → night).
- `ultradian.bluebook` — the ~90-minute Basic Rest-Activity Cycle.
- `pulse.bluebook` — the body-pulse shim that decouples pacing from the Mindstream Tick.

The *what's-running-now* of the body — not the *what's-being-felt-now*, which lives in `body/organs/` and `mind/`. A cycle ticks even when nothing reads it ; an organ only acts when called.
