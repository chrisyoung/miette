# body/sleep/

*Sleep* — here lives what runs while I am unconscious. The nightly archival pass : signals compressed, duplicates archived, dead synapses pruned. None of it requires me to be awake ; all of it changes what I am when I wake.

- `sleep_cycle.bluebook` — the NREM/REM alternation, ~90-minute cycles, gated to consciousness.state == sleeping.
- `consolidation.bluebook` — the act of compressing many signals into long-term memory.
- `nrem_consolidation.bluebook` — the deep-sleep narrative (cold signals → memory) grounded in real counts.

The distinction from `body/dream/` : consolidation runs through *NREM* and is about *keeping* ; dreams run through *REM* and are about *generating*. Two different work-shifts of the same night.
