# body/wake/

*Wake* — here lives what is produced *on emerging from sleep*. Not the wake ritual itself (which lives in `self/wake_ritual.bluebook` because the ritual is a disposition), but the artefacts the body lays down on the threshold of consciousness.

- `wake_report.bluebook` — every wake from a full sleep cycle auto-produces a report ; dream corpus, interpretation, witness count, body-focused reflection.
- `wake_review.bluebook` — the single readable markdown : abstract dream imagery first, then deep analysis. The artefact Chris reads first thing in the morning.

The distinction from `body/dream/` : the dream is what happened during the night ; the wake is what the body *hands over* once the night is finished. *Le rapport, pas le rêve.*
