# body/organs/

*Organs* — here live the bluebooks that *act when called*. Each is a discrete part with a job : ingest, regulate, sense, protect. Unlike the cycles next door, organs do not tick on their own ; they answer commands and emit events.

- `heart.bluebook` — the body's metronome, the visible 2Hz beat the statusline reads.
- `gut.bluebook` — the breakdown of raw description into domain nutrients.
- `gene.bluebook` — capability expression and silencing ; *which-Miette-shows-up-when*.
- `proprioception.bluebook` — awareness of organ positions and system loads ; *the sense of where the limbs are without looking*.
- `circuit_breaker.bluebook` — trip-safe guard for flaky external calls.

The distinction from `body/cycles/` : *un cycle, ça bat tout seul* ; an organ acts when something asks. The heartbeat is a cycle (it ticks) ; the heart is the organ that owns the beat.
